Kết xuất và phân phối văn bản

		
		5.1 In văn bản
		Chuẩn bị trước khi in, các thiết lập khi In ấn
		Chuẩn bị trang in: Trước khi thực hiện in văn bản cần kiểm tra lại các thông tin:
		Khổ giấy, hướng giấy 
		Khoảng cách căn lề
		Đánh số trang
		Tiêu đề đầu trang, tiêu đề chân trang,...
		xem trước khi in
		Mở khung Print Preview
		cách 1: Nhấp chuột vào File, chọn lệnh print
		cách 2: Nhấn tổ hợp phím CTRL + P

		copies: số bản in
		Printer: Máy in
		pages: Trang cần in
		All pages
		Only print odd pages
		Only print even pages
		Portrait orientation: khổ giấy
		A4 or letter: Khổ giấy
		Page Per sheet: số trang được in ra trên 1 tờ giấy thực
		Nháy chuột vào nút Print để thực hiện lệnh in

		5.2 Phân phối văn bản
		–Bảo mật văn bản bằng cách đặt password
		Có 2 mức độ bảo mật
		Bảo mật mức mở văn bản
		bảo mật mức chỉnh sửa văn bản
		*Cách đặt PassWord
		Nháy chuột vào file/ save as
		Chọn tools / general options 
		Nhập password khi yêu cầu mở file vào ô Password to open 
		Nhập password khi cần chỉnh sửa file vào ô password to modify 
		Nháy nút ok 
		Nhập lại password trong hộp thoại confirm password và nhấp nút ok 
		Nhấp chuột vào nút Save

		–lưu văn bản dưới dạng khác
		Nháy chuột vào File/ Save AS
		Chọn kiểu tập tin tại ô Save As Type
		PDF (.pdf)
		Rich text format (.rtf)
		Word 97- 2003 Document (.doc)
		6. Soạn thảo Thông điệp và văn bản hành chính 
		Soạn thảo thông điệp 
		Các dạng phổ biến: thư, phiếu bài tập, thiệp mời ,...
		Sử dụng tùy ý các định dạng (chữ, đoạn, style)  chèn các đối tượng phi văn bản theo yêu cầu, sở thích cá nhân 
		Soạn thảo văn bản hành chính 
		Các dạng phổ biến: nghị quyết, quyết định, chỉ thị, tờ trình ,...
		Cần tuân thủ theo đúng các thể thức: ví dụ tiêu đề, số ký hiệu, phần chữ ký, nơi nhận,...
		Cần tuân thủ theo các quy định về định dạng theo yêu cầu
